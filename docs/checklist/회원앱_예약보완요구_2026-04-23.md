## 3. 운영 해석 기준

- 회원이 직접 트레이너 시간을 선택한 예약은 `승인 대기`가 기본 상태다.
- 트레이너가 자신의 일정 화면에서 회원을 지정해 생성한 수업은 `즉시 확정`이다.
- PT/골프 레슨 횟수는 승인이나 일정 확정이 아니라 `수업 완료(completed)` 시점에만 차감한다.
- 회원의 확인 화면은 `내 예약 / 수업 이력`, `Golf 예약`, `이용권 / 잔여 회차`를 함께 본다.
- 트레이너의 처리 화면은 `일정 관리`, `회원 상세`를 중심으로 연결한다.

## 4. 현재 구현 해석

- 이번 정리는 `기획 문서 반영`이 목적이다.
- 현재 코드 기준으로는 회원 승인형 예약, 골프 예약 일정 캘린더, PT 차감 이력 / 다음 예약 일정, 트레이너 승인/거절, 트레이너 수업 이력 조회가 모두 완성된 상태는 아니다.
- 따라서 본 문서는 `기획 반영 완료`, `구현 후속` 기준으로 읽어야 한다.
