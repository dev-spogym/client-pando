## 해석 메모

- `포인트`는 `마일리지`로 통일했다.
- `회원가입`은 `CRM 등록 + 앱 연동`으로 정의했다.
- `개인정보·로그아웃`은 `설정 + 약관/정책 + 동의관리 + 탈퇴` 묶음으로 확장했다.
- 체크리스트는 모두 IA / 화면 / 기능 ID에 연결되도록 정리했다.
