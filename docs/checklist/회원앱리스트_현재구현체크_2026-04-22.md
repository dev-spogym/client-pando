# 회원앱 체크리스트 구현 점검

> 기준 파일: `docs/checklist/회원앱리스트_2025_1228_요약.xlsx`
> 점검 기준일: 2026-04-22
> 점검 대상: 현재 `client-pando` 프로젝트 (2026-04-22 골프 예약/결제/개인정보 보강 반영 후)

---

## 1. 판정 기준

- `구현`: 현재 프로젝트에서 퍼블리싱 화면과 핵심 사용 흐름이 가능
- `부분구현`: 화면은 있으나 체크리스트가 기대하는 일부 세부 기능/분기/정보가 비어 있음
- `미구현`: 현재 프로젝트에서 대응 화면/흐름이 없음

---

## 2. 요약

| 구분 | 건수 |
|---|---:|
| 전체 항목 | 37 |
| 구현 | 33 |
| 부분구현 | 4 |
| 미구현 | 0 |

핵심 변화:

- 골프 예약 화면에 `골프 강사 예약`과 `골프 타석 예약`을 함께 배치했다.
- `상품 스토어 → 상품 상세 → 결제 → 영수증`, `개인 결제`, `결제 내역` 흐름을 추가했다.
- `약관/정책`, `동의관리`, `회원 탈퇴` 화면을 설정 허브와 연결했다.

## 4. 남은 보강 포인트

1. PT 예약에서 `OT1/OT2` 타입을 명시적으로 분리하기
2. 마이페이지에서 회원 기본정보 직접 수정 화면 추가하기
3. 이용권 영역에 `락커 만료일`과 관련 상태 UI 추가하기
4. 레슨 이력을 `PT/GX/필라테스` 기준으로 더 명확히 분리하기
5. 이후 운영 단계에서 PG, 실예약, 리워드 원장을 붙일지 별도 결정하기

---

## 5. 체크리스트 외 추가 보완 요구

아래 항목은 기존 37개 체크리스트 판정과 별개로, 2026-04-23 추가 운영 요구로 분리 관리한다.

1. 골프 예약 후 회원 본인의 예약 일정을 캘린더/리스트로 다시 확인하기
2. PT 회원의 잔여 횟수와 차감 일자를 이용권/예약 화면에서 함께 보여주기
3. 회원이 트레이너 스케줄을 보고 원하는 시간에 예약 요청하기
4. 트레이너가 회원 요청 예약을 승인/거절하고 필요 시 시간 변경 제안하기
5. 트레이너가 자신의 캘린더에서 회원을 직접 지정해 수업을 추가하기
6. 트레이너가 담당 회원별 수업 이력과 승인 이력을 확인하기
