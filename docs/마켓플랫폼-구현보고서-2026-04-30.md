## 2. 디자인 시스템 변경 사항

### 2.1 BottomTabBar 재구성

기존 회원 탭바(홈 / 예약 / QR / 이용권 / MY)를 **마켓 5탭**으로 전환.

| 탭 | 경로 | 아이콘 | 비고 |
|---|---|---|---|
| 홈 | `/` | Home | MA-300 ExploreHome (둘러보기 홈) |
| 센터 | `/centers` | Building2 | MA-310 내 주변 |
| 강사 | `/trainers` | UserSquare2 | MA-320 강사 둘러보기 |
| 스크랩 | `/scrap` | Heart (filled when active) | MA-330 — 스크랩 카운트 배지 표시 |
| MY | `/profile` | User | 기존 마이페이지 |

기존 단일 센터 회원 흐름(예약/QR/이용권/수업이력)은 ExploreHome의 "내 센터" 카드 + MY 메뉴에서 진입.

### 2.2 PWA 설치 배너 위치 변경

- 기존: 하단 floating
- 변경: **상단 floating** + 새 디자인 시스템 적용 (rounded-card, shadow-card-elevated, Button 컴포넌트, X 닫기 아이콘)
- 추가 유틸리티: `mobile-floating-banner-top` 클래스 + `slideDown` 애니메이션

### 2.3 시설 태그 보강

기존 FACILITY_OPTIONS에 `골프 시뮬레이터`, `필라테스 룸`, `요가 룸`, `GX 스튜디오` 추가.

### 2.4 회원 로그인 에러 메시지 폴백 개선

`src/stores/authStore.ts` `login()`:
- Auth 실패 시 members 테이블 조회 → 회원이 존재하면 "앱 가입이 필요합니다" 안내
- 콘솔 디버그 로그 (`[Auth] signIn failed:`) 추가
