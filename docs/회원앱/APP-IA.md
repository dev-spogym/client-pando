# FitGenie 회원앱 정보구조 (IA)

| 항목 | 내용 |
|---|---|
| 버전 | 2.7 |
| 작성일 | 2026-04-23 |
| 상태 | 기획 유지 + 체크리스트 반영 + 퍼블리싱 기준 현재 구현 상태 병기 |
| 플랫폼 | Mobile App (iOS / Android / PWA) |
| 기준 | `회원앱리스트_2025_1228_요약.xlsx` + 현재 프로젝트 구조 |

## 1. 핵심 원칙

- 회원앱은 `회원 중심 통합 앱`이다.
- 회원가입은 `CRM 등록 + 모바일 앱 휴대폰 인증 연동` 방식으로 해석한다.
- 체크리스트의 `포인트`는 `마일리지`로 통일한다.
- `배지`, `알림센터`, `온보딩`, `강사 상세`, `대기 예약`, `수업 후기`, `재등록 추천`을 회원앱 현재 기획 범위에 포함한다.
- PT 플랜 외에도 `헬스장 이용권`, `골프장 이용권`, `골프 레슨권`, `개인 결제`를 동일 결제 도메인 안에 포함한다.
- 회원은 트레이너의 가능 시간과 자신의 예약 일정을 확인한 뒤 PT/골프 레슨을 요청할 수 있어야 한다.
- 회원이 생성한 트레이너 대상 예약은 `트레이너 승인 대기`를 기본으로 하고, 트레이너가 직접 회원을 지정해 만든 일정은 즉시 확정으로 처리한다.
- PT/골프 레슨 횟수 차감은 예약 승인이나 일정 확정 시점이 아니라 `수업 완료(completed)` 시점에 처리한다.
- 실제 Supabase/PG 연동이 없어도 회원이 흐름을 이해할 수 있도록 `퍼블리싱 화면`과 `모의 사용 흐름`을 제공한다.
- 화면설계서/기능명세서/IA는 동일한 화면 ID와 기능 ID를 사용한다.

## 2. 역할 구조

| Role | 설명 |
|---|---|
| member | 일반 회원 |
| trainer | 트레이너 |
| golf_trainer | 골프강사 |
| fc | FC |
| staff | 스태프 |

회원앱 체크리스트 반영 작업의 중심은 `member` 영역이다.  
트레이너/골프강사/FC/스태프는 기존 IA를 유지하되 회원 흐름과 연결되는 화면만 참조한다.

### 2.1 로그인 진입 구조

| 로그인 구분 | 입력 방식 | 선택 단계 | 실제 role | 기본 진입 경로 |
|---|---|---|---|---|
| 회원 로그인 | 전화번호 + 비밀번호 | 없음 | member | `/` |
| 직원 로그인 > 트레이너 | 아이디 + 비밀번호 | 트레이너 선택 | trainer, golf_trainer, admin | `/trainer` |
| 직원 로그인 > FC | 아이디 + 비밀번호 | FC 선택 | fc | `/fc` |
| 직원 로그인 > 스태프 | 아이디 + 비밀번호 | 스태프 선택 | staff | `/staff` |

`golf_trainer`는 직원 로그인 화면에서 `트레이너` 계열로 묶어 진입한다.

### 2.2 회원-트레이너 예약 연동 원칙

| 주체 | 진입 화면 | 동작 | 결과 상태 |
|---|---|---|---|
| 회원 | MA-125, MA-123 | 트레이너/골프강사 가능 시간 확인 후 예약 요청 | 승인 대기 |
| 회원 | MA-121 | 공개 수업 또는 즉시 예약형 수업 예약 | 즉시 확정 또는 대기 |
| 트레이너 | MA-210 | 자신의 캘린더에서 회원 지정 후 수업 직접 생성 | 즉시 확정 |
| 트레이너 | MA-210, MA-221 | 회원 요청 예약 승인 / 거절 | 확정 또는 거절 |

회원의 확인 화면은 `MA-122 내 예약 / 수업 이력`, `MA-123 Golf 예약`, `MA-131 이용권 / 잔여 회차`로 연결하고,  
트레이너의 처리 화면은 `MA-210 일정 관리`, `MA-221 회원 상세`로 연결한다.
예약 승인/직접 배정은 일정 확정 의미이며, 횟수 차감은 수업 완료 이후에만 반영한다.

## 6. 회원 탭 구조

```text
[ 홈 ] [ 예약 ] [ QR ] [ 이용권 ] [ MY ]
 MA-100  MA-120  MA-110  MA-131   MA-130
```

보조 화면은 탭 내부 스택으로 진입한다.

- 홈 하위: 리워드, 온보딩, 알림센터, 공지
- 예약 하위: 수업 상세, 골프 예약, 강사 상세, 대기 예약, 승인 대기, 후기
- 이용권 하위: 상품 스토어, 상품 상세, 결제, 재등록 추천, 결제 내역, 영수증
- MY 하위: 설정, 약관, 동의관리, 탈퇴, 리워드, 알림, 공지, 온보딩
- 예약/이용권 공통 보조 정보: 골프 예약 일정, PT 차감 이력, 다음 예약 일정, 트레이너 승인 상태

## 7. 주요 회원 플로우

### 6.1 CRM 회원 앱 연동 플로우

```text
CRM 회원 등록
  → 앱 설치
  → MA-002 앱 가입 / 연동
    → 이름/연락처 확인
    → 휴대폰 인증
    → 비밀번호 설정
  → MA-001 로그인
  → MA-127 온보딩 설문
  → MA-128 첫 루틴 제안
  → MA-100 홈
```

### 6.2 예약 / 대기 / 후기 플로우

```text
MA-100 또는 MA-120
  → MA-121 수업 상세
    → 공개 수업이면 예약 확정
    → 트레이너 승인형 수업이면 승인 요청 생성
    → 정원 마감 시 MA-124 대기 예약
    → 강사 보기 시 MA-125
  → MA-125 강사 상세
    → 가능 시간 확인
    → 회원이 원하는 시간으로 예약 요청
  → MA-122 내 예약 / 수업 이력
    → 승인 대기 / 확정 / 거절 상태 확인
    → 완료 수업 선택
    → MA-126 수업 후기
```

### 6.3 Golf 강사 예약 / 타석 예약 플로우

```text
MA-100 또는 MA-123
  → 골프 강사 예약 탭
    → 강사/시간대 선택
    → 예약 요청 또는 예약 확정
    → 내 골프 레슨 예약 일정 확인
  → 골프 타석 예약 탭
    → 타석 선택
    → 예약 확인
    → 만석 시 대기 등록
```

### 6.4 PT 잔여횟수 / 차감 이력 플로우

```text
MA-100 또는 MA-131
  → MA-131 이용권 / 잔여 회차
    → PT 이용권 선택
    → 총 횟수 / 사용 횟수 / 잔여 횟수 확인
    → 최근 차감 일자 확인
    → 다음 예약 예정 일자 확인(참고 정보)
  → MA-122 내 예약 / 수업 이력
    → 예약된 PT 일정과 연계 확인
  → 트레이너 수업 완료 처리
    → completed 전환
    → 해당 회차 차감 및 차감 이력 반영
```

### 6.5 상품 구매 / 개인 결제 플로우

```text
MA-131 또는 MA-134 또는 MA-141
  → MA-134 상품 스토어
    → MA-139 상품 상세
    → MA-140 결제하기
      → 카드 / 계좌이체 / 네이버페이 / 카카오페이 선택
      → 마일리지 사용
      → MA-142 영수증 상세
  → MA-141 개인 결제
    → 수동 결제 항목 생성
    → MA-140 결제하기
```

### 6.6 리워드 / 재등록 플로우

```text
MA-100 홈
  → MA-136 리워드 센터
    → 쿠폰 / 마일리지 / 배지 확인
  → MA-138 재등록 추천 플랜
    → 상품 결제 또는 결제 이력 확인
```

### 6.7 약관 / 동의 / 탈퇴 플로우

```text
MA-155 설정
  → MA-156 약관 / 정책
  → MA-157 동의관리
    → 마케팅 수신 상태 변경
  → MA-158 회원 탈퇴
    → 사유 선택
    → 추가 의견 입력
    → 탈퇴 요청 저장
```

## 8. 체크리스트 반영 메모

- `포인트`는 `마일리지`로 통일한다.
- 로그인은 `회원 로그인 / 직원 로그인`으로 나누고, 직원 로그인은 `트레이너 / FC / 스태프` 선택 구조로 유지한다.
- `배지`는 `리워드 센터` 내 별도 컬렉션으로 추가한다.
- `온보딩 설문`과 `첫 루틴 제안`은 `MA-127~128`로 독립시킨다.
- `회원가입`은 공개 회원 생성이 아니라 `CRM 등록 회원의 앱 연동`으로 정의한다.
- `결제수단 선택·결제`, `영수증`, `개인 결제`, `헬스장/골프장 이용권 구매`는 `MA-133~142` 영역으로 확장한다.
- `회원이 트레이너 스케줄을 보고 예약 요청`, `트레이너 승인 대기`, `골프 예약 일정 조회`, `PT 차감 이력 조회`는 `MA-121`, `MA-122`, `MA-123`, `MA-125`, `MA-131` 영역에서 함께 다룬다.
- `개인정보·로그아웃`은 설정 단일 화면이 아니라 `설정 + 약관/정책 + 동의관리 + 탈퇴` 묶음으로 해석한다.
