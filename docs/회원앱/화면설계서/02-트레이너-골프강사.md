# 스포짐 회원앱 화면설계서 — 트레이너 · 골프강사 (구현 기준)

> 작성일: 2026-04-22
> 기준 코드: `src/App.tsx`, `src/components/TrainerTabBar.tsx`, `src/components/TrainerRoute.tsx`, `src/stores/authStore.ts`
> 문서 목적: 현재 트레이너 화면 구현 상태 정리
> 범위: 트레이너 화면 5개 + 골프강사 확장 구현 상태
> 역할 체계: `trainer`, `admin`

---

## 1. 문서 기준

- 이 문서는 현재 저장소에 실제 존재하는 트레이너 화면만 정리한다.
- `authStore`는 현재 `member`, `trainer`, `admin`만 지원한다.
- 기존 기획 문서에 있던 `golf_trainer`, `fc`, `staff` 역할은 현재 앱 라우팅에 연결되지 않는다.

### 1.1 현재 지원 범위

- 트레이너 홈
- 회원 목록
- 회원 상세
- 일정 관리
- 수업 피드백

### 1.2 현재 미구현 범위

- 수업 상세 시작/완료 처리 전용 화면
- 노쇼/페널티 관리
- 수업 템플릿
- PT 횟수/완강 현황
- 강사 성과 대시보드
- 트레이너 전용 프로필/설정/알림
- 골프강사 전용 쌍방서명과 확인서

### 1.3 트레이너 탭 구조

| 탭 | 경로 | 구현 상태 |
|---|---|---|
| 홈 | `/trainer` | 구현 |
| 회원 | `/trainer/members` | 구현 |
| 일정 | `/trainer/schedule` | 구현 |
| 피드백 | `/trainer/feedback` | 구현 |
| MY | `/profile` | 미구현, 현재 `/trainer`로 리다이렉트 |

### 1.4 공통 동작 규칙

- `TrainerRoute`는 `trainer` 또는 `admin`만 허용한다.
- 트레이너가 회원 라우트(`/profile` 포함)에 접근하면 `PrivateRoute`가 `/trainer`로 리다이렉트한다.
- `/trainer/members/:id` 상세 경로에서는 하단 탭이 숨겨진다.

### 1.5 구현상 주의사항

- 홈의 `오늘 방문`은 담당 회원이 아니라 지점 전체 출석 수를 센다.
- `회원 관리`는 담당 회원이 아니라 지점 전체 회원을 조회한다.
- `공지작성` 퀵메뉴는 실제 작성 화면이 아니라 회원용 공지 조회 화면(`/notices`)으로 이동한다.
- `일정 관리`는 주간 보기와 수업 추가만 구현돼 있고 수정/삭제 기능은 없다.
- `수업 피드백`은 `classId`가 아닌 출석 시간대 기준으로 참여 회원을 추정한다.
- 골프강사 전용 역할, 쌍방서명, 확인서 화면은 현재 코드에 없다.

## 3. 화면 상세

## SCR-MA-200. 트레이너 홈

- 파일: `src/pages/trainer/TrainerHome.tsx`
- 목적: 트레이너가 오늘 수업과 현장 현황을 빠르게 본다.
- 주요 UI:
  - 트레이너 이름/역할 헤더
  - `오늘 수업`, `오늘 방문` 요약 카드
  - 다음 수업 카드
  - 회원관리/일정관리/운동피드백/공지작성 퀵메뉴
- 데이터:
  - `classes`: 오늘 날짜, 지점, `staffId` 기준 수업
  - `attendance`: 오늘 날짜, 지점 기준 출석 count
- 비고:
  - `오늘 방문`은 담당 회원이 아니라 지점 전체 출석이다.
  - 공지작성은 실제 작성이 아니라 `/notices` 조회 화면으로 이동한다.

## SCR-MA-210. 일정 관리

- 파일: `src/pages/trainer/TrainerSchedule.tsx`
- 목적: 주간 수업 일정 확인과 신규 수업 추가.
- 주요 UI:
  - 주간 이동 헤더
  - 요일 선택 7칸
  - 선택 날짜 수업 리스트
  - `수업 추가` 바텀시트
- 데이터:
  - `classes`를 지점/`staffId`/주간 범위 기준 조회
  - 수업 추가 시 `classes`에 직접 insert
- 주요 동작:
  - 월요일 기준 주 시작일을 계산한다.
  - 선택 날짜의 수업만 리스트로 필터링한다.
  - 추가 모달에서 제목, 타입, 날짜, 시간, 장소, 정원을 입력한다.
- 비고:
  - 월간/일간 뷰는 없다.
  - 수정, 삭제, 드래그 이동, 참석 처리 기능은 없다.

## SCR-MA-220. 회원 관리

- 파일: `src/pages/trainer/TrainerMembers.tsx`
- 목적: 회원 검색과 빠른 상세 진입.
- 주요 UI:
  - 검색 입력
  - `전체/오늘 방문/관심회원` 필터
  - 회원 카드 리스트
- 데이터:
  - `members`: 지점 전체 회원
  - `attendance`: 오늘 지점 출석 회원 ID 집합
- 주요 동작:
  - 이름/전화번호 검색
  - `오늘 방문` 필터는 오늘 출석한 회원만 노출
  - 카드 탭 시 상세 화면 이동
- 비고:
  - 담당 회원 개념이 아니라 지점 전체 회원을 보여준다.
  - `관심회원`은 `members.isFavorite` 플래그에 의존한다.

## SCR-MA-221. 회원 상세

- 파일: `src/pages/trainer/TrainerMemberDetail.tsx`
- 목적: 개별 회원의 운동, 체성분, 평가, 메모를 한 화면에서 조회/작성.
- 주요 UI:
  - 회원 기본 정보 카드
  - 탭 4개: `운동기록`, `체성분`, `평가`, `메모`
  - 평가 작성 폼
  - 메모 입력 필드
- 데이터:
  - `members`: 기본 회원 정보
  - `exercise_logs`: 운동기록
  - `body_compositions`: 체성분
  - `member_evaluations`: 평가
  - `member_memos`: 메모
- 주요 동작:
  - 탭 전환 시 필요한 데이터만 로드한다.
  - 평가 저장 시 `member_evaluations` insert
  - 메모 저장 시 `member_memos` insert
- 비고:
  - 별도 수정/삭제는 없다.
  - 회원 일정, 이용권 이력, 담당 배정 변경 같은 관리 기능은 없다.

## SCR-MA-223. 수업 피드백

- 파일: `src/pages/trainer/TrainerFeedback.tsx`
- 목적: 수업 단위로 참석 회원을 불러와 일괄 피드백을 저장한다.
- 뷰 구성:
  - 목록 뷰: 기간 필터(`오늘/이번 주`), 수업 카드 리스트
  - 작성 뷰: 참석 회원별 점수 슬라이더와 텍스트 입력
- 데이터:
  - `classes`: 지점/`staffId` 기준 수업
  - `attendance`: 수업 시간 구간 안의 출석 회원
  - 저장 대상: `member_evaluations`
- 주요 동작:
  - 수업 선택 시 시간대 기반 참석 회원을 중복 제거해서 가져온다.
  - 내용이 입력된 회원만 저장한다.
  - 저장 후 목록 뷰로 복귀한다.
- 비고:
  - `attendance`를 시간대 기반으로만 조회하므로 같은 시간대 다른 활동 회원이 포함될 수 있다.
  - 클래스 상세 화면과 연결된 피드백 진입은 없다.
