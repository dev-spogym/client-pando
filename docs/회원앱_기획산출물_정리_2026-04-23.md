# 회원앱 기획 산출물 정리

## 1. 산출물 목적

- 회원앱 관련 기획 업무 결과물을 항목별로 구분해 정리
- 보고 시 `기획 산출물`이 무엇인지 바로 확인할 수 있도록 정리

## 2. 기획 산출물 구성

### 2-1. IA

- 산출물명: `APP-IA`
- 목적:
  - 회원앱 전체 구조를 한눈에 보여주는 기준 문서
  - 역할별 메뉴 구조와 화면 흐름을 정리하는 문서

포함 내용:

- 역할별 주요 메뉴
- 주요 진입 흐름
- 화면 간 연결 구조
- 서비스 구조의 전체 윤곽

### 2-2. 기능명세서

- 산출물명:
  - `01-공통-회원`
  - `02-트레이너-골프강사`
  - `03-FC-스태프`

- 목적:
  - 역할별 기능 범위와 세부 항목을 정리하는 문서
  - 어떤 기능을 어떤 기준으로 봐야 하는지 정리하는 문서

포함 내용:

- 기능명
- 기능 목적
- 주요 사용 흐름
- 상태 구분
- 검토 포인트

### 2-3. 화면설계서

- 산출물명:
  - `01-공통-회원`
  - `02-트레이너-골프강사`
  - `03-FC-스태프`

- 목적:
  - 화면별 구성과 상태를 정리하는 문서
  - 실제 화면 검토 전에 확인 기준을 잡는 문서

포함 내용:

- 화면 구성 요소
- 화면별 동작 흐름
- 상태별 화면 기준
- 확인 포인트

### 2-4. 보조 기획 문서

- `APP-기획연계정의서`
- `APP-사용자시나리오`
- 회원앱 기획 개요 문서

목적:

- 기획 배경과 흐름을 보완 설명
- IA, 기능명세, 화면설계를 읽는 기준 제공

## 3. 기획 산출물 분류 결과

| 구분 | 수량 | 비고 |
|------|------|------|
| IA | `1종` | 전체 구조 기준 문서 |
| 기능명세서 | `3종` | 역할별 기능 기준 문서 |
| 화면설계서 | `3종` | 역할별 화면 기준 문서 |
| 보조 기획 문서 | `3종` | 기획 이해 보조 문서 |

## 4. 보고용 정리 문구

- 회원앱 기획 산출물은 `IA`, `기능명세서`, `화면설계서`를 중심으로 정리되어 있습니다.
- 따라서 본 프로젝트의 기획 업무 결과는 단순 보고서가 아니라, 실제 검토 가능한 기획 산출물 세트로 정리된 상태입니다.
