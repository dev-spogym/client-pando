# 회원앱 퍼블리싱 산출물 정리

## 1. 산출물 목적

- 회원앱 관련 퍼블리싱 업무 결과물을 보고용으로 정리
- 기획 산출물이 실제 화면 검토 자료로 어떻게 이어졌는지 구분해서 설명할 수 있도록 정리

## 2. 퍼블리싱 산출물 구성

### 2-1. 기본 구성

- 회원앱 주요 화면 퍼블리싱 정리
- 화면별 상태 검토 자료 정리
- 카테고리별 화면 묶음 정리

### 2-2. 포함 화면 범위

- 계정 진입
- 홈 / 출석 / 알림 / 공지
- 예약 / 수업 / 후기 / 대기예약
- 온보딩 / 운동 / 진단
- 이용권 / 결제 / 영수증 / 개인 결제
- 리워드 / 쿠폰 / 설정 / 동의 / 탈퇴

### 2-3. 포함 상태 범위

- 기본 화면
- 탭 상태
- 모달 / 바텀시트 상태
- 완료 상태
- 에러 상태

## 3. 퍼블리싱 산출물 특징

- 실제 회원앱 흐름을 기준으로 검토할 수 있도록 정리
- 단일 화면이 아니라 상태별 화면까지 함께 볼 수 있도록 정리
- 클라이언트 검토와 내부 피드백 수집에 바로 활용 가능한 형태로 정리

## 4. 퍼블리싱 산출물 결과

- 회원앱 주요 흐름 전반이 퍼블리싱 검토 범위 안에 포함되어 있습니다.
- 따라서 현재 퍼블리싱 산출물은 기획 내용을 화면 기준으로 설명하고 검토하는 데 사용할 수 있는 자료로 정리된 상태입니다.

## 5. 보고용 정리 문구

- 회원앱 퍼블리싱 산출물은 기획 내용을 실제 화면 검토 자료로 정리한 결과물입니다.
- 따라서 보고 시에는 `기획 산출물`과 별도로 `퍼블리싱 산출물`이 정리된 상태라고 설명하는 것이 적절합니다.
